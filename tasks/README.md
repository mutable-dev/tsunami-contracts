# Tasks

## Set up
```bash
node install
export CONTRACT=terra...
export LP_CONTRACT=terra...
```
## Query basket
```bash
node ./query_basket.js
```

## Query lp balance
```bash
node ./query_lp.js
```

## Provide liquidity
```bash
node ./provide_liquidity.js <denom> <amount>
```

## Swap
```bash
node ./swap.js <offer_denom> <offer_amount> <ask_denom>
```
